## 1. 크리에이터 자격 조건

크리에이터(DJ) 계정은 인디피 뮤직 운영팀의 **직접 심사 및 승인**을 통해서만 개설됩니다.  
자유 가입은 지원하지 않으며, 아래 조건을 충족하는 지원자에 한해 계정이 생성됩니다.

| 조건 | 기준 |
|------|------|
| 장르 적합성 | 매장 배경음악으로 적합한 장르 (Chill, Indie, Jazz, Lo-fi 등) |
| 음원 품질 | 업로드 가능 수준의 완성된 트랙 보유 |
| 저작권 | 본인이 직접 제작하거나 사용 권한을 보유한 음원만 등록 가능 |
| 활동 의지 | 최소 월 10트랙 이상 꾸준히 업로드 가능한 크리에이터 |

> 심사 기간은 신청일로부터 영업일 기준 1~5일이며, 결과는 등록 이메일로 통보됩니다.

---

## 2. 음원 등록 기준

업로드한 트랙이 아래 기준을 충족하지 않을 경우 정산 대상에서 제외되거나 삭제될 수 있습니다.

| 항목 | 기준 |
|------|------|
| 파일 형식 | MP3 또는 WAV |
| 음질 | 최소 128kbps 이상 (권장 192kbps 이상) |
| 트랙 길이 | 최소 2분 이상 |
| 파일 크기 | 트랙당 최대 50MB |
| 저작권 | 본인 소유 또는 사용 허락 받은 음원만 허용 |
| 콘텐츠 | 폭력적·선정적·혐오 표현 포함 음원 금지 |

## 3. 수익 정책

### 3-1. 수익 계산 방식

크리에이터 수익은 **구독 중인 사용자가 본인 음악을 실제 청취한 시간**을 기준으로 산정합니다. (구독자 수 기준 아님)

```
크리에이터 적립액 = (활성 구독자의 해당 크리에이터 음악 청취시간) × 1.04원

환급 단가 = 1.04원 / 청취시간 (= 0.01736원/분)
```

**단가 산출 근거 (크리에이터 1인 기준):**
```
구독자 2,000명 × 8시간 × 30일 = 480,000 청취시간
480,000시간 × 1.0417원 = 500,000원
```

> 결제액 4,400원에서 부가세(400원)·카드 수수료(150원)를 제외한 순정산액(약 3,850원)을 재원으로 하며,
> 본 단가는 순수익의 약 6.5% 수준입니다. 상세 산출은 [정산 정책서](CREATOR_SETTLEMENT_POLICY.md) 참조.

**본인 음악 청취 규모별 예상 적립 (일 8시간 기준):**

| 청취 규모 | 월 청취시간 | 월 예상 적립 |
|----------|-----------|------------|
| 50명 × 8h | 12,000h | 12,500원 |
| 100명 × 8h | 24,000h | 25,000원 |
| 500명 × 8h | 120,000h | 125,000원 |
| 1,000명 × 8h | 240,000h | 250,000원 |
| 2,000명 × 8h | 480,000h | 500,000원 |

> 구독자 1인의 청취 시간이 여러 크리에이터에게 분산될 경우, 각 크리에이터에게는 실청취 시간만큼 각각 적립됩니다.

### 3-2. 지속가능성 상한

전체 크리에이터 월 환급 총액이 **순수익의 7%를 초과**하면 단가를 재검토합니다.  
단가 변경은 **시행 30일 전 공지**하며, 이미 적립된 잔액은 소급 환수하지 않습니다. (변경은 향후 적립분에만 적용)

### 3-3. 청취 시간 측정 방식

- 구독자 브라우저에서 재생시간을 누적해 **5분(300초) 단위(또는 일시정지·트랙변경·이탈 시)** 서버로 전송
- 실제 재생된 시간만 누적 (일시정지, 탭 비활성화 시 제외)
- **활성 구독자**의 청취만 인정하며, 1인당 하루 **18시간**까지만 집계
- 비정상 패턴(동일 IP 다수 접속, 24시간 연속 청취, 자동 재생 봇 등)은 자동 제외

## 4. 정산 정책

정산은 **자동 지급이 아니라 적립 후 신청형**으로 운영합니다. 적립 잔액이 **5만원 이상**일 때 크리에이터가 직접 신청해야 지급됩니다.

### 4-1. 정산 일정

| 구분 | 내용 |
|------|------|
| 정산 기준 | 매월 1일 ~ 말일 청취 데이터 |
| 적립 반영 | 익월 10일 (월 마감 후 적립금 확정) |
| **최소 신청 금액** | 적립 잔액 **50,000원 이상** |
| 신청 방식 | DJ 대시보드 정산 카드에서 직접 신청 |
| 신청 마감 / 지급일 | 매월 말일 / 익월 15일 |
| 미달·미신청 | 다음 달로 자동 이월 (소멸 없음) |

### 4-2. 수익 확인

- DJ 대시보드 내 **정산 카드**에서 실시간 적립 잔액 및 이번 달 예상 적립 확인 가능
- 확정 적립금은 익월 10일 이후 업데이트되며, 5만원을 넘기면 신청 버튼이 활성화됩니다

### 4-3. 세금 처리

| 구분 | 처리 방식 |
|------|---------|
| 개인 크리에이터 | 정산액의 3.3% 원천징수 후 지급 |
| 사업자 크리에이터 | 세금계산서 역발행 후 지급 (VAT 별도) |

> 연간 수령액이 일정 기준 초과 시 종합소득세 신고 의무가 발생합니다. 세무 신고는 크리에이터 본인 책임입니다.

## 5. 크리에이터 의무 및 준수사항

### 5-1. 저작권

- 업로드하는 모든 트랙에 대해 **저작권 보유 또는 사용 허락**을 획득한 상태여야 합니다.
- 제3자 저작물 무단 사용으로 발생하는 법적 분쟁의 책임은 전적으로 크리에이터에게 있습니다.
- 저작권 문제가 확인된 트랙은 즉시 삭제되며, 해당 트랙으로 발생한 수익은 환수됩니다.

### 5-2. 콘텐츠 품질 유지

- 매월 최소 **5트랙 이상** 업로드를 유지해야 정상 크리에이터 지위가 유지됩니다.
- 3개월 연속 최소 업로드 미달 시 계정이 **휴면 상태**로 전환됩니다.
- 휴면 상태에서는 라이브 기능이 비활성화되며, 트랙 업로드 시 즉시 복구됩니다.

### 5-3. 라이브 방송 규칙

- 라이브 ON 시 반드시 청취 가능한 상태(트랙 1개 이상 적용 완료)여야 합니다.
- 라이브 중 페이지를 닫거나 방송을 종료할 때는 반드시 **방송 종료 처리**를 해야 합니다.
- 빈 방송(무음, 불완전한 트랙) 상태로 라이브 유지 시 경고 조치됩니다.

### 5-4. 계정 보안

- 계정(이메일, 비밀번호)을 제3자와 공유하거나 양도하는 행위를 금지합니다.
- 계정 공유로 발생한 어뷰징의 책임은 계정 명의자에게 있습니다.

## 6. 금지 행위

아래 행위가 확인될 경우 수익 무효 처리 또는 계정 제재가 적용됩니다.

| 금지 행위 | 내용 |
|---------|------|
| 어뷰징 | 봇, 자동 재생, 비정상 트래픽을 이용한 청취 수 조작 |
| 저작권 침해 | 타인 음원을 허락 없이 업로드 |
| 계정 공유 | DJ 계정을 타인에게 제공하거나 대리 운영 |
| 허위 정보 등록 | 실명, 연락처, 계좌 정보 허위 기재 |
| 부적절 콘텐츠 | 폭력, 혐오, 성인 콘텐츠가 포함된 음원 업로드 |
| 플랫폼 훼손 | 서버 공격, 크롤링, 비정상적 API 접근 |

---

## 7. 제재 정책

위반 행위의 경중에 따라 아래 단계로 제재가 적용됩니다.

| 단계 | 조건 | 조치 |
|------|------|------|
| 1단계 경고 | 경미한 위반 최초 1회 | 이메일 경고 발송 |
| 2단계 정산 보류 | 경고 후 재발 또는 중대 위반 | 해당 월 정산 보류 (조사 완료 후 지급 또는 환수) |
| 3단계 계정 비활성화 | 반복 위반 또는 심각한 위반 | 계정 영구 비활성화, 미지급 수익 환수 |

> 어뷰징으로 인한 비정상 청취가 확인된 경우 해당 기간 수익은 즉시 무효 처리됩니다.

---

## 8. 정책 변경 및 고지

- 수익 단가, Cap, 정산 일정 등 주요 정책 변경 시 **시행 30일 전** 이메일 및 대시보드 공지
- 크리에이터가 변경 정책에 동의하지 않을 경우 계정 탈퇴 신청 가능
- 탈퇴 시 확정된 미지급 수익은 마지막 정산일에 지급

## 9. 요약 카드

```
청취 단가          1.04원 / 청취시간 (= 0.01736원/분)
환급 기준          활성 구독자의 본인 음악 실청취시간
정산 방식          적립 후 신청형 (자동 지급 아님)
최소 신청 금액     50,000원 이상 (미달 시 다음 달 이월)
신청 마감 / 지급일 매월 말일 / 익월 15일
지속가능성 상한    순수익의 7% 초과 시 단가 재검토 (소급 환수 없음)
원천징수           개인 3.3% / 사업자 세금계산서
최소 업로드 유지   월 5트랙
휴면 전환 기준     3개월 연속 미달
```

> 정산 단가 산출 근거·적립/신청 절차·측정 기준·서버 비용 상세는 [CREATOR_SETTLEMENT_POLICY.md](CREATOR_SETTLEMENT_POLICY.md) 참조.
